# sublime-optimal-lines

Highlight characters to the optimal character count

## Getting Started
Install the module with: `npm install sublime-optimal-lines`

```javascript
var sublime_optimal_lines = require('sublime-optimal-lines');
sublime_optimal_lines.awesome(); // "awesome"
```
